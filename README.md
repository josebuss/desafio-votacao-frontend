# Desafio Votação --- Frontend

Frontend da aplicação de votação desenvolvido com **React**.
Este projeto consome a API do backend responsável por gerenciar pautas,
sessões de votação e votos.


------------------------------------------------------------------------

## 🚀 Tecnologias utilizadas

-   React
-   Next.js
-   TypeScript
-   TailwindCSS
-   shadcn/ui
-   Lucide Icons

------------------------------------------------------------------------

## ⚙️ Instalação

Clone o repositório:

``` bash
git clone https://github.com/josebuss/desafio-votacao-frontend.git
```

Entre na pasta do projeto:

``` bash
cd desafio-votacao-frontend
```

Instale as dependências:

``` bash
npm install
```

------------------------------------------------------------------------

## ▶️ Executando o projeto

Inicie o servidor de desenvolvimento:

``` bash
npm run dev
```

A aplicação ficará disponível em:

http://localhost:3000

------------------------------------------------------------------------

## 🔗 Integração com Backend

O frontend consome os endpoints disponibilizados pelo backend:

https://github.com/josebuss/desafio-votacao

Certifique-se de que a API esteja rodando para que as funcionalidades da
aplicação funcionem corretamente.

------------------------------------------------------------------------

## 🧪 Funcionalidades

A aplicação permite:

-   Criar **pautas**
-   Abrir **sessões de votação**
-   Registrar **votos**
-   Visualizar **resultados das votações**
